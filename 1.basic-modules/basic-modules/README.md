# 기초 모듈

![](../../immersion\_days\_logo.png)

기본 모듈에서는 각 AWS 기본 서비스의 다양한 기능을 배울 수 있습니다.

기본 모듈은 다음과 같은 내용으로 구성됩니다.

* 컴퓨트 - Amazon EC2
* 네트워크 - Amazon VPC
* 보안 - AWS IAM
* 모니터링 - Amazon CloudWatch
* 데이터베이스 - Amazon RDS
* 스토리지 - Amazon S3

각 실습을 통해, 서비스의 특성에 대해 알아보겠습니다.
